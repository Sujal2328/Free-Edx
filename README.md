# Free-Edex 🌟

Free-Edex is a structured and completely free learning platform designed to help students follow a clear roadmap instead of getting lost in scattered online resources.

## 📚 About the Project

Free-Edex provides curated, high-quality, and 100% free learning paths for:

* JEE Preparation (Mains & Advanced)

All resources are handpicked and organized to reduce confusion and help students focus on learning effectively.

## 🚀 Features

* Structured roadmap for JEE preparation
* Topic-wise best free video lectures and notes
* Daily, weekly, and monthly study plans
* Clean and distraction-free interface
* No paid promotions
* Beginner-friendly guidance
* Integrated chatbot for student support and doubt assistance

## 💡 Mission

To make quality education accessible, structured, and free for every student.

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Chatbot Integration

## 📂 Project Status

This project is currently under development and will continue to improve with more structured learning paths and enhanced chatbot support.
